# 🎯 BetForge — Casa de Apostas React + Tailwind

Interface premium de casa de apostas esportivas com React + Vite + Tailwind CSS.

## 🚀 Como Rodar

```bash
npm install
npm run dev
# Acesse: http://localhost:5173
```

## 📱 Screens

| Rota | Descrição |
|------|-----------|
| `/` | Página Inicial |
| `/live` | Jogos Ao Vivo |
| `/sports` | Todos os Esportes |
| `/history` | Histórico de Apostas |
| `/promotions` | Promoções |
| `/account` | Minha Conta |

## ✨ Funcionalidades

- Boletim de apostas (simples e múltipla)
- Odds ao vivo com flutuação em tempo real
- Filtros por esporte e liga
- Histórico com resumo de ganhos
- Perfil com stats e configurações
- Design dark mode premium

## 🛠 Stack

React 18 · Vite · Tailwind CSS 3 · React Router DOM · Lucide React
